# Quality Assessment Rubric (High Threshold)

## Pass Threshold: 80.0

## Criteria

### Accuracy (weight: 3)
- All facts and claims must be verifiable and correct
- No misleading or ambiguous statements
- Sources properly referenced where applicable

### Clarity (weight: 2)
- Clear and logical structure
- Accessible language appropriate for the target audience
- Smooth transitions between sections

### Completeness (weight: 1)
- All required topics covered
- Sufficient depth of analysis
- No critical gaps in coverage
